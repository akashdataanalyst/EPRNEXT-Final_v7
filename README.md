# calco_erp

Calco PolyTechnik Pvt Ltd ERP custom app for ERPNext / Frappe.

## Phase 1 scope

- Commercial approval gatekeepers
- Product development gatekeepers
- Vendor approval gatekeepers
- Shared dependency validation utilities

## Initial DocTypes

- Customer Master Approval
- NSL Record
- NPD Request
- NPD Trial Record
- Product Release Approval
- NVD Request
- Vendor Qualification Record
